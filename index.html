⚾ Pitching
Max pitches
75 per day. Pitcher reaching limit mid-count may finish that batter.
40-pitch rule
40+ pitches = ineligible to pitch NEXT game.
Rest days
1-20: 0 days · 21-35: 1 · 36-50: 2 · 51-65: 3 · 66+: 4
Catcher-Pitcher
4+ innings catching = can't pitch. 41+ pitches = can't catch.
3 HBP
Must be removed after hitting 3 batters.
🏃 Base Running
Steals
1 per half-inning. No stealing home. Double steals count as 1.
Overthrow
Only 1 base on an overthrow.
No dropped 3rd strike
Batter is out automatically.
Head-first slides
Advancing = out. Return to base = OK.
📊 Game Rules
6-run rule
Max 6 runs per half-inning. No limit in 6th/last inning.
Mercy
10-run lead after 4 innings (3½ for home).
Time
No new inning after 1:45 if another team waiting.
Forfeit
Can't field 8 = 5-0 forfeit.
👦 Playing Time
Minimum fielding
6-inn game: 3 innings. 4-inn: 2 innings.
CBO
Everyone bats. Order is fixed. Player removed mid-game = automatic out each time spot comes up (LL 4.04(h)).
10 fielders
Extra outfielder if 10+ players.
No bunting
Bunting not permitted.
