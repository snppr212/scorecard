Run attribution gaps

For each game, compares the opponent's total runs (what our pitchers should be charged) against the sum of runs charged to our pitchers, and against the sum of G.runs entries. A gap means runs scored that were never logged as attributable runs (manual score bumps, or runners that scored but weren't recorded in pscored). Also dumps e4 + e9 half-innings (the biggest suspects) so we can see which runners scored. Read-only.
