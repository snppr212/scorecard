Fix inherited-runner attribution

Baseball rule: a run is charged to the pitcher who was on the mound when that runner reached base, not whoever's pitching when they cross the plate. This tool replays every finalized game, matches each scoring runner to the AB where they reached base, and re-charges the run (and ER) to the correct pitcher.

How it matches: for each run in G.runs, it finds the earliest un-consumed at-bat in the same half-inning where that batting slot reached base (1B/2B/3B/HR/BB/HBP/ROE/FC/CI), and uses that AB's pitcher. The unearned flag on each run is preserved, so ER stays correct. Preview first; nothing writes until you click Apply.
