Audit every game — what we actually have

Read-only. Walks every finalized game and reports the raw data so you can see what's trustworthy and what isn't. No assumptions, no fabrication, no writes.

Each game shows:
  • fullPitchLog flag — explicit per-game flag (v208). True = user confirmed per-pitch logging.
  • Per-AB stats — total AB count + result breakdown
  • Per-AB pc consistency — how many ABs have pitches.length matching pc vs not
  • Pitch-code histogram — counts of B/S/L/F/FT/X/H/U/Us across all ABs
  • Per-pitcher — name, stored total (PC), stored IP, count of ABs they're attributed to, sum of pc across those ABs (flagged if mismatch)
  • Per-inning — for each half, which pitcher(s), AB count, sum of pc
